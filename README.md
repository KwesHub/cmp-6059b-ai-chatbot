# CMP-6059B AI Chatbot  Group Project

Advanced Artificial Intelligence coursework (CW2).
A conversational chatbot for a train operating company built with
Python, spaCy, experta, Keras, SQLite, and Streamlit.

## Project Structure
```
cmp-6059b-ai-chatbot/
├── app.py                  # Streamlit UI — entry point, run this
├── database.py             # SQLite setup and connection
├── requirements.txt        # Install all dependencies from here
│
├── nlp/
│   ├── intent.py           # spaCy intent recogniser (book_ticket / predict_delay)
│   └── entities.py         # spaCy entity extractor (stations, dates, times)
│
├── engine/
│   ├── engine.py           # experta KnowledgeEngine class
│   ├── rules.py            # @Rule dialogue flow functions
│   └── knowledge_base.py   # KB Q&A content and fallback rules
│
├── task1/
│   ├── ticket_api.py       # National Rail / Brfares API call
│   └── station_codes.py    # Station name to CRS code mapping
│
├── task2/
│   ├── clean_data.py       # Merge and clean the 4 CSV files
│   ├── preprocess.py       # Feature engineering and train/test split
│   ├── train_models.py     # Train kNN, linear regression, random forest
│   ├── keras_model.py      # Keras MLP training and export
│   └── predict.py          # Model loader used by the chatbot
│
├── data/                   # NOT in GitHub — get CSV files from group Drive
└── models/                 # Saved model files generated by train_models.py
```

## Setup: do this once

1. Clone the repo:
   git clone https://github.com/KwesGithub/cmp-6059b-ai-chatbot.git
   cd cmp-6059b-ai-chatbot

2. Install dependencies:
   pip install -r requirements.txt

3. Download the spaCy language model:
   python -m spacy download en_core_web_sm

4. Set up the database:
   python database.py

5. Run the chatbot:
   streamlit run app.py

## Data files

The CSV files (2022-2025_service_details.csv) are NOT in this repo.
Download them from Blackboard and put them in the data/ folder.
